# Job Market Analysis

## Project Overview
This project analyzes job market data to identify trends in job roles, skills demand, and salary distribution using Power BI.

## Project Workflow
1. Raw data collection
2. Data cleaning and preprocessing
3. Analysis and visualization in Power BI
4. Insights presentation using PowerPoint

## Folder Structure
- **01_Raw_Data**: Original dataset (unchanged)
- **02_Cleaned_Data**: Cleaned and processed dataset
- **03_PowerBI_Dashboard**: Power BI dashboard file
- **04_Presentation**: Final presentation (PPT)
- **05_Screenshots**: Dashboard images

## Tools Used
- Power BI
- Excel / CSV
- Power Query
- PowerPoint

## Key Insights
- High demand for data-related roles
- Salary trends vary by experience and location
- Certain technical skills are consistently in demand
